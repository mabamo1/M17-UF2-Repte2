### Exercici 6
Jo he escollit Godot (què és un motor de videojocs open-source).

Godot permet, tant amb GDScript (el seu llenguatge de programació) com amb C# (C Sharp), programar amb l'estil de programació orientada a objectes (OOP: Object-Oriented Programming).

De fet, a l'hora de crear scripts, per programar qualsevol cosa, el propi script automàticament hereda d'una altra classe pare.
